# Python Password Generator

A secure and customizable password generator written in Python.  
This tool allows users to generate strong random passwords using letters, numbers, and special characters for improved security.

---

## Features

- Generate strong random passwords
- Custom password length
- Option to include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Fast and lightweight
- Simple command-line interface

---

## Installation

Clone the repository:

git clone https://github.com/tariq-almalky/password-generator.git

cd password-generator

Run the script:

python PasswordGenerator.py

---

## Usage

Run the program:

python PasswordGenerator.py

Follow the prompts to select:

- Password length
- Character types to include

The program will generate a secure random password.

---
